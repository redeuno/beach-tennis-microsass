# Beach Tennis Arena - MicroSaaS Project

## Estrutura do Projeto

Este projeto contém a documentação e desenvolvimento de um microsass para gestão de arena de beach tennis.

### Organização das Pastas

- `docs-analysis/` - Documentos originais para análise
- `versions/` - Versões dos documentos criados
- `current/` - Versão atual dos documentos principais
- `assets/` - Recursos e imagens do projeto

### Versionamento

Todos os documentos seguem o padrão de versionamento semântico:
- v1.0 - Versão inicial
- v1.1 - Melhorias menores
- v2.0 - Mudanças significativas

### Como Usar

1. Faça upload dos documentos para análise na pasta `docs-analysis/`
2. Os documentos processados serão salvos em `versions/` com numeração
3. A versão mais atual sempre estará em `current/`

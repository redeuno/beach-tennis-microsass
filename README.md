# Beach Tennis MicroSaaS

Sistema de gestão para beach tennis desenvolvido com Kiro AI Assistant.

## Estrutura do Projeto

- `assets/` - Recursos e imagens
- `current/` - Versão atual do sistema
- `docs-analysis/` - Documentação e análises
- `versions/` - Histórico de versões

## Tecnologias

- Sistema MicroSaaS para gestão de beach tennis
- Interface moderna e responsiva
- Gestão de jogadores, torneios e quadras

## Como usar

Consulte a documentação específica na pasta `docs-analysis/` para mais detalhes.
